## What's been built (v0.5.0)

Everything the user asked for is now in the JSX:

- **Automated peak classifier** with cluster analysis. Clusters nearby peaks, annotates each with nearest predicted species, reports main peak + relative sizes + relative abundances per cluster, plus channel-level abundance.
- **Dye mobility offset correction** — per-dye state with manual entry, reset, and auto-calibration from tallest peak's closest blunt prediction.
- **Generalization** — editable construct sequence textarea with target-range inputs, validation, reset-to-V059 button.
- **Cross-dye interpretation** — pairs (B,Y) and (G,R) clusters and interprets Δ as blunt / 1-4 nt overhang / missing-adapter.
- **Tutorial** — `docs/TUTORIAL.md` with concrete V059_3-2 and gRNA3_1-1 walkthroughs, line-number index into the code.
